Register Account with CRYT-id

*Click Associate account to Activate CRYT-id Login System to your account.

**For Developers: This random generate 99 charset passphrase for this account and create a token from it.
Remember to save passphrase to CRYT-id Table associated with this account if need to give CRYT Wallet Function.
*Email is basic required on most website content, if need more input, like userid, or phone add it to form and handle on destination POST page.
**passphrase is hidden if you need to manage userswallet on your website, do it visible for allow users to have passphrase available to send and receive transactions.
